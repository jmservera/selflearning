## Key Decisions

1. **Vite + React 18 + TypeScript** — Fast builds, type safety, ecosystem maturity
2. **Tailwind CSS** — Small bundle, dark mode built-in, no external component lib
3. **Centralized API client** — Namespaced endpoints prevent drift from backend contracts
4. **Custom hooks** — Data fetching logic isolated, reusable across components
5. **Lazy loading for Oracle's pages** — Parallel work doesn't block Tank's build
6. **WebSocket auto-reconnect** — Exponential backoff (max 10 attempts) handles network issues
7. **Nginx proxy config** — API proxy + WebSocket upgrade headers + SPA routing fallback
